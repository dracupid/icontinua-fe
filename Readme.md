健康云服务前端（静态资源）
===========

[爱康体项目介绍](http://cdnst.icontinua.com/html/intro.html)

## 主要功能
- 体检报告历史查询和趋势分析
- 身体基础数据、血压、血氧、生物电详细测量结果、分析建议、用户排名
- 个人信息的编辑与修改
- 化验单上传识别

## 相关内容
- gulp: javascript项目管理工具
- React.js:
    + ant.design: React组件库
    + react-router: 页面路由
- webpack: 编译打包工具
- npm: nodejs包管理
- stylus: CSS预编译
- jade: HTML模板
- ECMASCRIPT 2015 / ECMASCRIPT 7, babeljs
- echarts: 图表绘制

## 相关脚本
- 编译(除三方库): `gulp build`
- 完整编译: `gulp`
- 实时预览: `gulp watch`
- 代码审查: `npm run lint`
- 自动修正代码样式: `npm run fix`
- 部署到苏州开发机: `./deploy-dev.sh`
- 部署到阿里云: `./deploy-production.sh`
